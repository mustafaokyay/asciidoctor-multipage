Custom Level Part

This second part has multipage-level=1 set, which causes level-1 nodes to be leaf nodes (split into chapters). Nodes above that level are branch (or root) nodes, while nodes below that level are content nodes.

The first part, About This Sample, has no multipage-level attribute set, so it uses the document-level attribute, which is multipage-level=0 (split into parts).
